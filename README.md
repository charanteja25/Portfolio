# Charan Gunisetty — Portfolio

Personal portfolio website for Charan Gunisetty, Data & AI Scientist with 6+ years of experience.

**Live site:** https://charanteja25.github.io

- `/` — Main portfolio (Hero, About, Projects, Experience, Skills, Writing, Contact)

Single-file HTML/CSS/JS. No framework, no build step. Deployed via GitHub Pages.
